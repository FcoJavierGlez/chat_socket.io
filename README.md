### Despliegue:

Hay que tener instalado [Node.js](https://nodejs.org/es/) en una versión 12+.

**Clonamos el proyecto:**
```
git clone https://github.com/FcoJavierGlez/chat_socket.io .
```

**Instalamos las dependecias:**
```
npm install
```

**Iniciamos:**
```
npm start
```
